Build with [Eleventy](https://www.11ty.dev/) and hosted on [Netlify](https://www.netlify.com/).

## Codespace development

    npm install
    npx @11ty/eleventy --serve

## Local development (with Docker)

    ./console start
    ./console npm install
    ./console serve
